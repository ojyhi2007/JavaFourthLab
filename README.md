## Задание 1.3 и 1.4  — Интерфейс сравнения и сравнение студентов 
<details>
<summary>Подробнее об задании</summary>
  
Интерфейс MyComparable и сравнение студентов

Описание:
Создан интерфейс MyComparable, который определяет общий способ сравнения объектов. На основе этого интерфейса класс Student реализует возможность сравнивать студентов между собой по их среднему баллу.

Логика решения:
Создан обобщённый интерфейс MyComparable<T>, содержащий метод compare(T other).
Метод возвращает целое число, показывающее результат сравнения объектов.

Класс Student реализует интерфейс MyComparable<Student>.
Сравнение выполняется на основе среднего балла студентов.

Метод compare() использует метод getAverageGrade() для вычисления среднего значения оценок и сравнивает два значения с помощью Double.compare().

Если средний балл текущего студента больше — метод возвращает положительное число.
Если средние баллы равны — возвращается 0.
Если средний балл меньше — возвращается отрицательное число.

Пример:
Создаются два студента:
Вася с оценками 3,4,5,4 и Петя с оценками 5,5,5,5.

## Задание 2.3 - Обобщённая коробка (создание класса Box<T> для хранения объектов)
<details>
<summary>Подробнее об задании</summary>
  
Обобщённая коробка Box<T>

Описание:   
Создан обобщённый класс Box<T>, который предназначен для хранения одного объекта произвольного типа. Класс позволяет поместить объект в коробку, достать его из коробки и проверить, пуста ли коробка. Дополнительно реализована работа с трёхмерной точкой Point3D через wildcard.

Логика решения:   
Класс Box<T> содержит одно поле типа T, в котором хранится объект.
Метод isEmpty() проверяет, находится ли объект в коробке.
Метод put() помещает объект в коробку, но только если она пуста. Если в коробке уже есть объект, выбрасывается IllegalStateException.
Метод take() возвращает объект из коробки и очищает коробку, то есть после извлечения ссылка на объект обнуляется.

Для демонстрации работы wildcard создан отдельный метод, принимающий коробку вида Box<? super Point3D>. Такой тип позволяет передавать коробки, которые могут хранить объект Point3D или его родительский тип. Внутри метода создаётся случайная трёхмерная точка, которая затем помещается в коробку.

Таким образом в задании показывается использование: обобщённого класса; параметра типа T; wildcard ? super Point3D; безопасной работы с объектом внутри контейнера.

Пример:
Создаётся коробка Box<Object>.
В неё с помощью отдельного метода добавляется случайная точка Point3D.
После этого объект выводится на экран, затем извлекается методом take(), и коробка снова становится пустой.

</details>
